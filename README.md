Iris Flower Classification (CodeAlpha Internship)
A clean, production-ready Python implementation developed during my Virtual Data Science Internship at **CodeAlpha**. This repository showcases multi-class classification fundamentals by predicting Iris flower species based on physical dimensions.

## 📌 Project Overview
The system takes measurements of an Iris flower's petals and sepals as input and accurately classifies it into one of three distinct species:
* *Iris-setosa*
* *Iris-versicolor*
* *Iris-virginica*

## 🛠️ Tech Stack & Libraries
* Language:** Python 3.x
* Machine Learning Library:** Scikit-Learn
* Data Manipulation: NumPy, Pandas
* UI/Core Execution: Built ready for interactive terminal/script inputs.
